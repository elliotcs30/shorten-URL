# shorten-URL

## 專案功能
* 使用者可以使用短網址連向原始網站
* 使用者可以按 Copy 來複製縮短後的網址
* 若使用者沒有輸入內容，就按下了送出鈕，需要防止表單送出並提示使用者

## 開始使用
1.確認已安裝Node.js  
2.將本專案clone進本地端
3.安裝套件
```js
npm i express@4.17.1 express-handlebars@3.0.0 mongoose@6.0.4 
```
4.執行程式
```js
npm run start
```
5.當cmd出現以下這行表示啟動成功
```js
App is running on http://localhost:3000
```
6.接著在瀏覽器輸入
```js
http://localhost:3000
```
